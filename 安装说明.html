安装 KeepFunChat

准备步骤

请先确保你能够访问国内互联网且关闭了 VPN 或 代理

Linux 用户需要具有 root 权限

Android 用户安装步骤

第一步：下载 ZeroTermux

Android 用户请首先下载 ZeroTermux。你可以通过以下链接获取：

  • GitHub 官方下载：ZeroTermux-0.118.1.41 版本
  • 国内高速镜像：国内用户推荐使用此链接

第二步：配置环境并安装 KeepFunChat

在安装完 ZeroTermux 后，请按照以下步骤操作：

  1. 打开 ZeroTermux。
  2. 复制以下命令并粘贴到终端中：
export GIT_SSL_NO_VERIFY=true repository="https://kkgithub.com/Yeah114/KeepFunChat" && cd ~ && sed -i 's@^\(deb.*stable main\)$@#\1\ndeb https://mirrors.tuna.tsinghua.edu.cn/termux/termux-packages-24 stable main@' $PREFIX/etc/apt/sources.list && apt update && apt upgrade -y && apt update && apt install git -y && git clone "$repository" ~/KeepFunChat && cd ~/KeepFunChat && bash ./setup.sh

Linux 用户安装步骤

Debian 系统

  1. 打开终端
  2. 复制以下命令并粘贴到终端中：
export GIT_SSL_NO_VERIFY=true repository="https://kkgithub.com/Yeah114/KeepFunChat"
cd ~
sudo apt install apt-transport-https ca-certificates
sudo sed -i 's@^\(deb.*http://deb.debian.org/debian\)$@#\1\ndeb https://mirrors.tuna.tsinghua.edu.cn/debian/ stable main@' /etc/apt/sources.list
sudo apt update
sudo apt upgrade -y
sudo apt install git -y
git clone "$repository" ~/KeepFunChat
cd ~/KeepFunChat
bash ./setup.sh

Ubuntu 系统

  1. 打开终端
  2. 复制以下命令并粘贴到终端中：
export GIT_SSL_NO_VERIFY=true repository="https://kkgithub.com/Yeah114/KeepFunChat"
cd ~
sudo apt install apt-transport-https ca-certificates -y
sudo sed -i 's@http://archive.ubuntu.com/ubuntu/@https://mirrors.tuna.tsinghua.edu.cn/ubuntu/@g' /etc/apt/sources.list
sudo apt update
sudo apt upgrade -y
sudo apt install git -y
git clone "$repository" ~/KeepFunChat
cd ~/KeepFunChat
bash ./setup.sh
